# Internship Portal - Flutter Mobile Application

## Project Overview
Internship Portal is a Flutter-based mobile application designed to connect students with internship opportunities across various industries. The app provides a user-friendly interface for discovering, applying to, and tracking internships, helping students gain valuable real-world experience efficiently.

## Features
- Browse and search for internship openings
- View detailed company and internship information
- Apply to internships directly through the app
- Track application statuses
- Personalized recommendations based on user profile
- User authentication and profile management

## Technologies Used
- Flutter & Dart for cross-platform mobile app development
- Local storage or in-app state management for data persistence (e.g., using Provider, Riverpod, or simple state management solutions)
- No backend or external database used; all data is handled locally within the app

## Installation
1. Clone the repository
   git clone https://github.com/prasanna2705/internship-portal.git
2. Navigate to the project directory
   cd internship-portal
3. Install dependencies
   flutter pub get
4. Run the app
   flutter run
